# 👋 Minimal
> Minimal is a landing page kit UI template built by [cssninjaStudio](https://cssninja.io).

[![cssninja-discord](https://img.shields.io/discord/785473098069311510?label=join%20us%20on%20discord&color=6944EC)](https://go.cssninja.io/discord)

## ✌️ preview

Check out the live demo by clicking [here](https://minimal.cssninja.io/). 
Minimal is built with [Bulma](https://bulma.io) and [Alpine JS](https://github.com/alpinejs/alpine).

## 👍 Features

* Astro v1.x
* Nodejs v16.x
* Bulma v0.9.x
* Alpine v3.x

## 👌 Usage

1. Install Depedencies

```sh
pnpm i
```

2. Run in dev mode

```sh
pnpm dev
```

3. Or build source

```sh
pnpm build
```
